# Electrical Panel Documentation & QR Management System

Phase 1 foundation for a Vue 3 + TypeScript enterprise frontend.

## Scripts

- npm run dev
- npm run build
- npm run lint
- npm run typecheck

## Phase 1 Scope

- Vue 3, TypeScript, Vite, Pinia, Vue Router, Axios, TanStack Query, PrimeVue, PrimeFlex, PrimeIcons
- Enterprise folder structure
- Responsive app shell with sidebar, header, breadcrumbs, global loading, and global error surfaces
- API, service, store, composable, validator, type, and utility layers
